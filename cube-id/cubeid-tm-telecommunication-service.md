# CubeID™ Telecommunication Service

CubeID™ provides telecommunication and data-transmission services through the CubeID™  hardware and supported firmware. This service enables wireless transmission of UAV identification and related telemetry information from unmanned aircraft to ground-based receivers, applications, and monitoring systems.

Using its integrated Bluetooth module, CubeID™  continuously broadcasts identification and telemetry data from the unmanned aircraft to CubePilot-operated cloud servers. This wireless data transmission forms the core of the CubeID™ telecommunication service.

### Key Service Features

#### Wireless Transmission of Telemetry-Related Data&#x20;

CubeID™ wirelessly transmits Remote ID data—including UAS ID, operator ID, and other applicable fields—from the UAV to ground-based applications and cloud services.

This information is displayed on CubePilot’s DroneShare website: [https://droneshare.cubepilot.com/#/drone/map](https://droneshare.cubepilot.com/#/drone/map)

#### Remote Monitoring Capabilities

Operators can view the UAV’s presence, identification, and certain flight parameters through the CubeID™  telecommunication and data-transmission services in real-time.

#### Service Available without Additional Subscription Fees

The broadcasting and reception of Remote ID data through CubeID™  hardware and supported firmware—along with its display on the DroneShare website—are provided as part of the CubeID™  telecommunication and data-transmission services, without any additional subscription charges.
